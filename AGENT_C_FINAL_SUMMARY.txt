================================================================================
                        DATABASE INTEGRATION
================================================================================

Dexie IndexedDB Schema:

TABLE: logCategories
  - id (string, primary key)
  - name, description, color
  - isDeleted (boolean, soft delete flag)
  - createdAt, updatedAt (ISO-8601)

TABLE: logEntries
  - id (string, primary key)
  - categoryId (foreign key to logCategories)
  - note (string)
  - date (string, YYYY-MM-DD format)
  - isDeleted (boolean, soft delete flag)
  - createdAt, updatedAt (ISO-8601)
  - Indexes: id, categoryId, date, createdAt

TABLE: notes
  - id (string, primary key)
  - title (string)
  - richContent (any, Quill Delta JSON)
  - isArchived (boolean)
  - isDeleted (boolean, soft delete flag)
  - createdAt, updatedAt (ISO-8601)
  - Indexes: id, createdAt, updatedAt

TABLE: tags
  - id (string, primary key)
  - name (string, unique per implementation)
  - color (string, optional)
  - createdAt, updatedAt (ISO-8601)
  - Indexes: id, name

TABLE: noteTags
  - Compound key: [noteId + tagId]
  - Junction table for many-to-many relationship

SEED DATA:
  - Default log categories with colors
  - Created on first app load via initializeWithSeed()
